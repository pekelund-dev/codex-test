About this starter

This Spring Boot application demonstrates a modern responsive UI coupled with Spring Security.

Technology stack

  • Spring Boot, Spring MVC, and Thymeleaf
  • Spring Security with OAuth2 client support
  • Bootstrap 5 for responsive components
  • Maven with Java 21 for modern language features

Next steps

Extend the starter with your business logic, connect to a database, or integrate additional OAuth2 providers.
